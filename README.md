# 🌐 Portfólio Pessoal

Este projeto consiste em um **portfólio pessoal**, com o objetivo de reunir informações profissionais, habilidades técnicas e projetos desenvolvidos ao longo da formação.

O portfólio foi construído utilizando tecnologias fundamentais do desenvolvimento web e está hospedado por meio do **GitHub Pages**.

---

## 🎯 Objetivo do Projeto

- Apresentar meu perfil profissional de forma organizada e visual
- Consolidar conhecimentos em **HTML, CSS e JavaScript**
- Aplicar conceitos de design responsivo e interatividade
- Utilizar o **GitHub** como plataforma de versionamento e publicação

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – estrutura das páginas  
- **CSS3** – estilização e layout  
- **JavaScript** – interatividade e comportamento da aplicação  
- **GitHub Pages** – hospedagem do projeto
